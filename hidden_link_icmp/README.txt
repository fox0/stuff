# hidden_link_icmp

Довольно успешная попытка написать сразу несколько вещей:
* демона
* сниффер трафика
* скрытый канал связи
* с ассемметричным шифрованием (!)
* подмена ip-адреса в исхоящих пакетах
* и передача информации в недоступную сеть путём отражения ICMP-трафика
от общедоступного сервера.
